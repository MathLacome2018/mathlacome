A sharp, uppercase action element. Use for CTAs, form submits, and navigation triggers.

```jsx
<Button variant="primary" size="md">Read More</Button>
<Button variant="ghost" size="sm">Subscribe</Button>
<Button variant="text">← Back</Button>
```

Notable variants: `primary` (black fill, inverts on hover), `ghost` (outline, fills on hover), `text` (no border, dims on hover). Sizes: `sm`, `md`, `lg`. Pass `href` to render as an anchor.
